# A1-WAVE-Space-Agent

This repository contains code for the A1 WAVE Space Agent project.

Files:
- `space_agent_v2.py` - main agent script
- `run_agent.ps1` - helper PowerShell script
- `watchlist.json` - example watchlist

Instructions:
1. Review `.env` and fill in `GITHUB_TOKEN` if you want automated pushes.
2. Initialize git and push to your GitHub repository `zubeir/Trading-AI-Agents` under the `A1-WAVE-Space-Agents` folder.

Note: Do not commit real tokens to public repositories. Use GitHub Secrets or local environment variables in CI.
